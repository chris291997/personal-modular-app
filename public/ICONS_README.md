# PWA Icons

You need to create the following icon files for the PWA:

1. `pwa-192x192.png` - 192x192 pixels
2. `pwa-512x512.png` - 512x512 pixels
3. `apple-touch-icon.png` - 180x180 pixels (optional, for iOS)
4. `mask-icon.svg` - SVG icon (optional)

You can:
- Use an online icon generator
- Create them in a design tool
- Use a placeholder generator

Place all icons in the `public/` folder.

For now, the app will work without these icons, but they're needed for a proper PWA installation experience.
